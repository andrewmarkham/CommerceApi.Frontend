## Usage

```ts
import { AddToCart } from '@jhoose-commerce/components';


const Example = () => {
  return (
      <Drawer heading="Your Cart" show={showDrawer} close={() => setShowDrawer(false)}>
        ...
      </Drawer>
  );
};

export default Example;
```

#### Arguments

- **heading** _ Heading for the drawer.
- **show** _ show/hide the drawer.
- **close** - A function that is called when the close button is clicked.
